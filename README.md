# OpenGL Starter
A solid starting point for new opengl projects. 

Fetches glfw, glad, glm, stb, and tinyobjloader using CMake. 

Includes basic mesh, texture, shader and material support.

## How to build
Open or navigate command prompt to the root folder containing 'CMakeLists.txt' and run this command

`cmake -B build`

Or just run build.bat, located in the root folder.
